# vue-picture-card

> A vue component to show a picture card

## Installation
``` bash
npm install vue-picture-card
```

## Usage
``` javascript
import VuePictureCard from 'vue-picture-card'
import 'vue-picture-card/dist/vue-picture-card.min.css'
export default {
  ...
  components:{
      "vue-picture-card": VuePictureCard
  },
  ...
}
```
